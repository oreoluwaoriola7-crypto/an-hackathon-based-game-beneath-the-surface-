# an-hackathon-based-game-beneath-the-surface-
an hackathon based game with theme beneath the surface 
Beneath the Surface is a browser-based 2D exploration game where you pilot a deep-sea diver descending through the ocean's most mysterious and dangerous zones. Collect bioluminescent orbs, manage your oxygen supply, dodge deadly sea creatures, and race to recover the Ancient Relic buried at 1000 meters below.
The game is built entirely with HTML5 Canvas, Vanilla JavaScript, and CSS — no frameworks, no dependencies, no build tools require
